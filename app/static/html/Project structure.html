Structure¶

apps¶

  • __init__.py

    Initialize the webapp.

  • webapp.py

    Start data steaming by linking to spotify. Sign in and sign out.

  • assets

    All stylesheets, images, and javascript should be in an assets folder

  • dashapp1

    • callbacks.py

      Callback functions for updating layouts. app.dashapp.callbacks

    • Layout.py

      Deploy layouts .

  • templates

    HTML files that define Layouts.

  • util

    • data_callbacks.py

      Callback functions for processing data. app.util.data_callbacks

    • Kmean.py

      Predict genres with K-means. app.util.K-means

    • recommender.py

      Generate recommended songs. app.util.recommender

    • spotify_interface.py

      Login, create cache, and retrieve data. app.util.spotify_interface

models¶

Trained machine learning models.

dashapp.py¶

Run to launch the app.
